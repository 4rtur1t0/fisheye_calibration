You can find an example at calibrate.sh

USAGE:
./camera_calibration calib_config.xml

All configuration parameters for the calibration are stored at calib_config.xml
Images for cameras are stored at calib_images_front.xml or calib_images_back.xml

Please, edit calib_config and change accordingly.

All parameters are written to out_camera_data.xml

To calibrate for fisheye/pinhole camera, change the setting at calib_config.xml
